# Week07 Vanilla JavaScript Assignment

## 프로젝트 구성

### `Week7_miniquest`

JavaScript 응용 교재의 미니퀘스트 풀이

### `community_frontend`

Vanilla JavaScript로 이벤트 처리를 먼저 구현한 후,
Fetch API를 적용하여 Spring Boot 백엔드와 연동

### `community_vdom`

Vanilla JavaScript로 V-DOM의 `h`, `render`, `patch` 함수를 구현하고,
Hash Router와 `hashchange` 이벤트를 이용해 SPA 형태로 확장

## 백엔드
